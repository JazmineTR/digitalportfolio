# Jazmine Le - Digital Portfolio

## Overview

Hey! This is my personal portfolio website, built to showcase projects relating to CS and writings which so far have been poems. 

## Features
- **Motion Effects** powered by Framer Motion
- **Responsive Design** with Tailwind CSS
- **Icons and UI components** from Shadcn and Radix

## Tech Stack
- **React.js** – Frontend framework for building UI components
- **Tailwind CSS** – Utility-first styling framework for responsive design
- **Framer Motion** – Animation library for React
- **Shadcn** – UI component library
- **Radix** - Icon library
